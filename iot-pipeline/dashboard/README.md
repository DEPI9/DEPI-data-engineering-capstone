# Dashboard

## Overview
This module contains notes and files related to the IoT analytics dashboard, which visualizes real-time temperature, humidity, and alert statistics using Power BI.

## Tools
- Power BI Desktop
- Azure SQL Database

## Included Visuals
- Temperature over time
- Humidity over time
- KPIs (Average Temp, Average Humidity)
- Alerts Count
- Normal vs Alert Distribution
