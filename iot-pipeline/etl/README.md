# ETL

## Overview
This folder contains the batch ETL script used to clean and transform the raw IoT sensor data generated by the generator module. The cleaned data is exported to a new CSV file.

## Files
- batch_etl.py — Cleans and transforms raw CSV data.
- cleaned_sensor_data.csv — Resulting cleaned dataset.

## Requirements
Install dependencies:
  pip install -r ../requirements.txt

## How to Run
python3 batch_etl.py

## Output
Cleaned CSV dataset stored as:
cleaned_sensor_data.csv
